Sobre o projeto
Esse site mostra a trajetória dos alimentos do campo até a nossa casa. Nele, o usuário pode conhecer mais sobre a produção de alimentos comuns do nosso dia a dia.

Objetivo
O site tem o objetivo de informar o leitor sobre a importância do agro, mostrando a trajetória de três alimentos principais: o Arroz, o Milho e o Chocolate.

Funcionalidades
Botões interativos para descobrir a história de cada alimento.
Exibição dos textos na tela de forma dinâmica.
Modo escuro para mudar o fundo (Acessibilidade).

Justificativa
Esse site foi desenvolvido para valorizar o trabalho do campo e mostrar as informações de forma acessível aos usuários, destacando como a produção sustentável equilibra a produção e o meio ambiente.

Instruções do Site
Início: Apresenta o tema e os três alimentos disponíveis.
Interação: O usuário escolhe um alimento e clica no botão "Ver mais".
Impacto: O site exibe logo abaixo a trajetória e curiosidades sobre o alimento selecionado.
